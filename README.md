This is my very own music/media player. You can play your own songs here by dragging and dropping .mp3 files into the provided space.
```
1. Drag a file from your file manager.
2. Drop the file onto the box.
3. Music plays automatically!
```
To navigate the song, there are the Back, Play and Next buttons to do so!
